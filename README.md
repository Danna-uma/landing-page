# 🌐 Landing Page - Escuela Platanar

## Descripción
Landing page desarrollada para la Escuela Platanar con el objetivo de presentar información institucional de forma clara, moderna y accesible. 
El proyecto se enfoca en el diseño responsivo y la organización visual del contenido.

---

## Tecnologías
- HTML5
- CSS3
- JavaScript
- Git y GitHub
- Responsive Design

---

## Funcionalidades
- Diseño moderno e institucional
- Secciones informativas organizadas
- Navegación fluida entre apartados
- Diseño adaptable a dispositivos móviles
- Interfaz limpia y fácil de usar

## Proyectos Destacados
- Landing Page Escuela Platanar (este proyecto)

- Sistema Java con Base de Datos

- Sistema de Clínica en C++
- Sistema de Inventario
